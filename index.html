About This Explorer

The Iroko Ewé Knowledge Explorer provides a public interface for navigating plant records structured using the Iroko Framework.

This environment demonstrates a Sacred Metadata approach: botanical and linguistic knowledge is discoverable while spiritually sensitive knowledge remains governed through explicit access tiers.

The dataset follows the CARE Principles for Indigenous Data Governance: Collective Benefit, Authority to Control, Responsibility, and Ethics.
