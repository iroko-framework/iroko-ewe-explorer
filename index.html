About This Explorer

This interface presents linked data generated via OpenRefine RDF Transform. It provides a browseable view of plant records, multilingual labels, and controlled vocabulary relationships within the Iroko Framework.

This explorer is read-only and does not expose restricted knowledge. Access tiers defined in the vocabulary are respected.
